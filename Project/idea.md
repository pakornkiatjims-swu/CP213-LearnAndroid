# Project: Edumension (Education + Dimension)
**Concept:** เปลี่ยนการเรียนรู้ให้เป็นการผจญภัยในโลกแฟนตาซี RPG

---

## MVP Phase: English Mastery
เน้นไปที่การเรียนภาษาอังกฤษในช่วงต้นเพื่อทดสอบระบบ Core Mechanic

### 1. ระบบวัดระดับ (Placement Test / Class Selection)
* **Initial Quest:** แบบทดสอบสั้นๆ ที่มาในรูปแบบการต่อสู้กับมอนสเตอร์ตัวแรก เพื่อวัดระดับพื้นฐาน (A1 - C2)
* **Result:** ผลคะแนนจะถูกแปลงเป็น "Base Stats" เช่น Vocabulary (STR), Grammar (DEF), Listening (AGI)

### 2. ระบบภารกิจ (Missions & Questlines)
* **Main Quest:** บทเรียนตามระดับของผู้ใช้ (เช่น การแต่งประโยค, คำศัพท์พื้นฐาน) เมื่อผ่านจะปลดล็อกเนื้อหาถัดไป
* **Daily Quests:** ภารกิจรายวันเพื่อสร้างวินัย เช่น "เรียนรู้คำศัพท์ใหม่ 5 คำ" หรือ "ฝึกพูด 3 ประโยค"
* **Side Quests:** ภารกิจเสริมเพื่อเก็บแต้มเพิ่ม เช่น การอ่านบทความสั้นๆ หรือการทำ Quiz ท้าทาย

### 3. ระบบเลเวลและไอเทม (Progression & Loot)
* **Level Up:** เมื่อได้ XP ครบจะอัปเลเวล เพื่อปลดล็อกสกิลการเรียนรู้ใหม่ๆ หรือชุดแต่งตัวละคร
* **Item Drops:** เมื่อทำภารกิจสำเร็จ จะมีโอกาสได้รับไอเทม เช่น
    * **Common Drop:** เหรียญทอง (Gold) สำหรับซื้อของในร้านค้า
    * **Rare Drop:** "การ์ดคำศัพท์ระดับแรร์" หรือ "หินตีบวก" (ใช้เพื่อเพิ่มพลังในการทำ Quiz)
* **Farming System:** ระบบทบทวนความรู้เดิม (Spaced Repetition) โดยเปรียบเทียบเป็นการ "กลับไปฟาร์มมอนสเตอร์ในจุดเดิม" เพื่อรับทรัพยากรมาอัปเกรดตัวละคร

## Next Phase: Expansion
การขยายขอบเขตเมื่อระบบ Core นิ่งแล้ว

### 1. Multi-Subject Integration
* เพิ่มหมวดวิชาอื่นๆ เช่น Coding, Mathematics, หรือ Science โดยใช้ระบบเลเวลแยกตามวิชา แต่ใช้ตัวละครเดิม
* **Cross-Skill Synergy:** การเรียนบางวิชาอาจช่วยเสริมพลังให้อีกวิชาได้ (เช่น เรียน Math เพิ่มจะช่วยให้เรียน Coding เก่งขึ้นในเกม)

### 2. AI & Smart System
* **AI Dungeon Master:** ระบบวิเคราะห์จุดอ่อนของผู้ใช้และสร้าง "ภารกิจเฉพาะบุคคล" เพื่อแก้จุดอ่อนนั้นโดยอัตโนมัติ
* **Dynamic Difficulty:** ปรับความยากของมอนสเตอร์ (โจทย์) ตามความเร็วและอัตราการตอบถูกของผู้ใช้ในขณะนั้น

### 3. Social & Guild System
* **Party Quest:** จับกลุ่มกับเพื่อนเพื่อทำโปรเจกต์หรือตอบคำถามยากๆ ร่วมกัน
* **Leaderboard:** การจัดอันดับนักผจญภัยในแต่ละสัปดาห์

## Tech Stack (Suggested)
* **Frontend:** Flutter หรือ React Native (เพื่อให้ใช้งานได้ทั้ง iOS/Android)
* **Backend:** Firebase หรือ Node.js (รองรับระบบ Real-time และ Gamification Logic)
* **Database:** PostgreSQL (สำหรับเก็บข้อมูลไอเทมและ Progression ที่ซับซ้อน)
